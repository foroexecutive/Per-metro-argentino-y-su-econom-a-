# Perímetro-argentino-y-su-economía-
El perímetro del país es de aproximadamente 15.000 km (14.493). Dos tercios lo forman las fronteras terrestres con cinco países y, el resto, el litoral marítimo comprendido por el Río de la Plata. El litoral marítimo de la Antártida Argentina e Islas Australes totaliza poco más de 11.000 km (11.235).
